# Welcome to A8n Docs

Welcome to **Autonomous (A8s)** — your self-service DevOps platform that lets developers deploy, monitor, and manage applications on Kubernetes — without ever touching infrastructure.

With A8s, you can connect your Git repository, deploy instantly, and let the platform handle infrastructure, scaling, and delivery.

---

## Why Autonomous?

Modern infrastructure is powerful — but complex. Kubernetes, Helm, ArgoCD, Vault, Prometheus — each tool is excellent alone, but stitching them together takes weeks of engineering effort.

Autonomous abstracts all of that into a single, developer-friendly platform. You connect your Git repository, click deploy, and your application is live. No YAML wrangling. No pipeline configuration. No infrastructure expertise required.

---

## Who Is This For?

- **Developers** who want to ship fast without waiting on DevOps teams
- **Students** learning cloud-native development without the overhead
- **Teams** who need consistent, repeatable deployment workflows
- **Engineers** who want Infrastructure as Code without the manual setup

---

## What You Can Do With Autonomous?

- Deploy any application from a Git repository in one click
- Monitor CPU, memory, and pod health from a personalized dashboard
- View real-time logs streamed directly from Kubernetes pods
- Manage secrets securely with Vault integration
- Roll back any deployment to a previous version instantly
- Scale applications automatically based on traffic load
- Manage container image artifacts through an integrated Harbor registry
- Deploy across multiple environments: dev, staging, production
